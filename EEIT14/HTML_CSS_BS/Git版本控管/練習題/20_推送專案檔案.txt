目標 : 建立 Lab_B 數據庫，提交並推送第一個版本


請依照下方步驟，在 GitHub 建立新的數據庫，並將 template 資料夾內的檔案推送上去。
※ 完成後的結果，如圖「20_完成圖.png」所示。


10. 請在 GitHub 頁面建立新的數據庫:「 Lab_B 」

20. 開啟 命令提示字元 | 終端機
    
    21. 在 命令提示字元 | 終端機，確認目前所在的資料夾

    22. 輸入 cd Desktop ，切換到「桌面」

    23. 使用「 git clone 數據庫位址 」指令，將 Lab_B 複製(下載)到「桌面」

30. 將 template 資料夾內的檔案手動放到 Lab_B 資料夾 準備推送

40. 使用 Visual Studio Code 開啟 Lab_B 資料夾

50. 輸入「 git add . 」將檔案放到暫存區列管

60. 輸入 git commit -m "提交第一版" ，提交檔案

70. 輸入 git log --oneline ，查看版本紀錄

80. 輸入 git push ，推送到 GitHub 數據庫

90. 在 GitHub 頁面查看數據庫變化
