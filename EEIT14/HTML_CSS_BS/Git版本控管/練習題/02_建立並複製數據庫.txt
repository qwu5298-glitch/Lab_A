目標 : 建立 GitHub 數據庫，並複製(下載)數據庫到本機


10. 建立 GitHub 數據庫

    11. 前往 GitHub 官網並登入帳號 (https://github.com/)

    12. 在 dashboard 頁面點擊「Create repository」或「New」按鈕 建立數據庫

    13. 在 Repository name 輸入框，輸入數據庫名稱:「 Lab_A 」

    14. 其餘選項維持預設，點擊最下方「Create repository」按鈕建立


20. 複製(下載) GitHub 數據庫到本機桌面。以下步驟請在「命令提示字元 | 終端機」輸入指令

    21. 請在 命令提示字元 | 終端機 確認目前所在的資料夾。
        - 命令提示字元: 請觀察「 > 」符號前面的路徑
        - 終端機: 輸入「 pwd 」指令，執行後查看顯示的路徑

    22. 使用指令「cd Desktop」切換到「桌面」

    23. 使用指令「git clone 數據庫位址」複製(下載) Lab_A 數據庫
